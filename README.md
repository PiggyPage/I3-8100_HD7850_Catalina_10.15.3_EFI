# I3-8100_HD7850_Catalina_10.15.3_EFI
## 配置
| CPU | 主板 | 内存 | 显卡 | 网卡 |
|  ----  | ----  | ----  | ----  | ---- |
| i3-8100 | 铭瑄H110M | 2*8GDDR3-1333专用条 | 迪兰HD7850-2G | 腾达U3 |

EFI十分稳定，升级自行更新CLover

无线网卡在腾达官网有10.15的无线网卡驱动，实测稳定跑满20M带宽
